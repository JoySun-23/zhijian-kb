# Changelog

## v2.0 — AI PM 专项版 (2024-03)

### 新增
- **AI PM 能力雷达**：六维自评图，点击滑条实时调分，追踪成长曲线
- **热点追踪面板**：Agent / 物理AI / 多模态 / 推理模型 / MCP，含趋势标签和有效期
- **能力维度标签**：技术/产品/商业/反模式/输出，贯穿卡片、图谱、编辑器
- **知识有效期**：为每篇笔记设置保质期，防止知识库熵增
- **Prompt 模板库**：CRISPE / CoT / PRD / 竞品分析 / 价值转化公式
- **知识图谱维度着色**：节点颜色反映能力维度，而非随机颜色
- **反模式案例文件夹**：专门收录 AI 产品翻车案例
- **localStorage 持久化**：刷新页面不丢数据
- **维度权重 AI 推荐**：相似推荐新增维度匹配权重

### 改进
- 卡片新增维度标签显示
- 日历时间轴新增维度色标
- 编辑器新增维度选择器和有效期选择
