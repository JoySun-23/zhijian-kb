快速上手

五步，从零到第一条知识入库

整个流程不超过 10 分钟。建议先用示例笔记感受一下，再开始录入自己的内容。

1
打开应用，浏览示例笔记
直接用浏览器打开 zhijian-v3.html，无需安装、无需服务器。首次打开会自动加载 6 篇示例笔记，涵盖 Agent、面经、Coze文档、转型方法论等主题，可以直接点开感受交互。
💡 数据存在浏览器 localStorage，换浏览器或清缓存会丢失。建议定期导出，或部署到 GitHub Pages 后使用。
2
配置 API Key，解锁 AI 自动提炼
点击右上角 ⚙ 设置，填入你的 Anthropic API Key（格式：sk-ant-…）。Key 仅存储在本地，不会上传任何服务器。配置后，每次剪藏内容时 Claude 会自动提炼摘要、要点和标签。
🔑 没有 API Key 也可以正常使用，只是剪藏时没有 AI 提炼功能，内容仍会正常保存。可以去 console.anthropic.com 申请。
# 设置弹窗里填入：
API Key: sk-ant-api03-xxxxxxxx
默认来源: 网页 / 小红书 / Coze
3
用「快速剪藏」录入第一条内容
点击顶部蓝色「快速剪藏」按钮（或按 ⌘D），选择内容来源，将你今天在小红书看到的面经、Coze 推送的学习文档，或者任意文章段落粘贴进去，点击「AI 处理并保存」。
粘贴文字 网页URL Coze 文档 Markdown
4
建立双链，让知识互相关联
打开任意笔记进入编辑模式，在正文中输入 [[ 加上另一篇笔记的标题，系统会自动高亮为可点击的链接。或者点击编辑器顶部的「⌁ 链接」按钮插入。
✦ 建议每篇笔记至少建立 1-2 个双链。链接越多，知识图谱越有价值，AI 推荐的准确度也会提高。

查看雷达图，做月度能力自评
点击顶部「雷达」视图，可以看到六维能力评估图。点击每条进度条调整当月分值（1-10分）。随着笔记积累，下方「知识库维度分布」柱状图会实时反映你在哪个方向的积累更多。建议每月初做一次自评。

使用技巧

让知识库越用越有价值的四个习惯

✦ 习惯一
每篇笔记至少建立 1 个双链
孤立的笔记是「知识坟场」。即使只是写 [[相关笔记名]] 一句话，也比没有链接强十倍。AI 相似推荐的准确度与双链密度正相关。
→ 习惯二
热点笔记设「热点」有效期
AI 领域变化极快。Agent 的最佳实践三个月可能就过时了。设置有效期后，卡片会显示保质期提醒，定期做「过期笔记清理」，防止知识库熵增。
⚠ 习惯三
每遇 AI 翻车案例，立即归入「反模式」
反模式库是 AI PM 最独特的竞争力之一。失败案例比成功案例更有价值——面试时能讲出「我知道这个方向的坑」，比只会说「我知道怎么做」印象深刻得多。
⬡ 习惯四
「输出」维度倒逼学习质量
每隔 2-3 周，把某个知识点整理成「输出」维度的笔记——哪怕只是一篇 200 字的心得。写出来的过程会暴露你真正不理解的地方，远比单纯收集有价值。
常见问题

FAQ

我的数据存在哪里？会丢失吗？▼
数据存储在浏览器的 localStorage 中，不会上传任何服务器，完全本地化。但这也意味着：清除浏览器缓存、换浏览器、或换设备时，数据会丢失。建议将 html 文件上传到 GitHub，并定期通过「导出」功能备份笔记内容。
没有 API Key 可以用吗？▼
完全可以。没有 API Key 时，剪藏功能仍然正常工作——内容会直接保存，只是不会自动生成 AI 摘要和标签。所有其他功能（双链、图谱、雷达、日历、相似推荐等）均不依赖 API Key。
API Key 安全吗？会被上传吗？▼
API Key 只存储在你本地浏览器的 localStorage 中，且只在你主动点击「AI 处理并保存」时才会直接从浏览器向 Anthropic API 发起请求。不经过任何中间服务器，代码完全开源可审计。
能从 Coze 自动同步吗？▼
目前 Coze 没有开放供个人使用的知识库导出 API，所以暂时需要手动复制粘贴。流程只需 2 分钟：打开 Coze → 全选复制 → 粘贴到「Coze 文档」标签 → AI 处理。后续若 Coze 开放 Webhook，可以配置自动推送，实现真正的自动化。
手机上可以用吗？▼
可以。应用已做移动端适配：侧边栏变为左上角汉堡菜单呼出的抽屉，编辑器在手机上全屏展示，剪藏抽屉也是全屏模式。最佳体验是用手机浏览器打开部署在 GitHub Pages 上的版本，方便随时收藏内容。
如何部署到 GitHub Pages？▼
1. 在 GitHub 创建新仓库（如 zhijian-kb）；2. 将 zhijian-v3.html 重命名为 index.html 后上传；3. 进入仓库 Settings → Pages → Source 选 main 分支 → Save；4. 约 1 分钟后访问 https://你的用户名.github.io/zhijian-kb。每次 git push 自动更新，免费且永久。
知识图谱节点位置每次都变？▼
是的，目前知识图谱使用简化的环形布局（链接数最多的节点居中，其余节点分布在环上），每次渲染会有微小随机偏移。完整的力导向图（Force-directed graph）在后续版本中规划实现，届时节点位置会稳定下来。
部署到 GitHub

三分钟，让知识库在线随时可用

GitHub Pages 免费托管，每次更新推送即自动部署。手机也能访问，再也不用找 html 文件了。
