# 纸间 · AI PM 个人知识库

> 一款面向「技术转型 AI 产品经理」的个人知识管理工具，米白纸质风格，双链笔记 + 知识图谱 + AI 能力雷达。

![版本](https://img.shields.io/badge/版本-v2.0-rust) ![纯前端](https://img.shields.io/badge/纯前端-单文件-teal) ![无依赖](https://img.shields.io/badge/依赖-无-green)

## ✨ 功能特性

| 功能 | 描述 |
|------|------|
| 📝 双链笔记 | `[[笔记名]]` 语法，自动建立反向链接 |
| 🧠 知识图谱 | 节点按能力维度着色，链接粗细表示关联度 |
| 📡 AI PM 雷达 | 六维能力自评，点击滑块实时更新，追踪成长轨迹 |
| 🔥 热点追踪面板 | Agent / 多模态 / 推理模型 / MCP 等前沿话题，含有效期标注 |
| 📅 日历视图 | 月历 + 时间轴双视图，按写作日期浏览知识积累 |
| ✦ AI 相似推荐 | 基于标签、维度、双链、内容关键词的相似度计算（含维度匹配权重） |
| ⌁ Prompt 模板库 | CRISPE / CoT / PRD / 竞品分析 / 价值转化公式，一键复制 |
| ↑ Markdown 导入 | 拖拽批量导入 `.md` 文件，自动解析标题、正文、双链 |
| ⏳ 知识有效期 | 为每篇笔记标注保质期（长期/1年/3月/热点），防止知识库熵增 |
| 📱 移动端适配 | 抽屉式侧边栏，全屏编辑器，触控友好 |

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/你的用户名/zhijian-kb.git

# 直接用浏览器打开（无需任何服务器）
open zhijian-v2.html
```

或者使用 **GitHub Pages** 一键部署：
1. 仓库 → Settings → Pages
2. Source 选择 `main` 分支 / `root` 目录
3. 保存后访问 `https://你的用户名.github.io/zhijian-kb`

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘K` / `Ctrl+K` | 聚焦搜索框 |
| `⌘N` / `Ctrl+N` | 新建笔记 |
| `⌘S` / `Ctrl+S` | 保存当前笔记 |
| `Esc` | 关闭编辑器 / 关闭弹窗 |
