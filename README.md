## 这是什么

纸间是一款**纯前端单文件**的个人知识库应用，无需安装、无需服务器，直接用浏览器打开即可使用。

专为从事 AI 相关工作、或正在向 AI 产品经理方向转型的技术人设计：

- 你在小红书刷到了面经 → 一键剪藏，AI 自动提炼要点
- Coze 每天推送的学习文档 → 粘贴导入，自动整理成知识卡片
- 零散的技术笔记 → 通过 `[[双链]]` 互相关联，形成知识网络
- 每月做一次能力自评 → 雷达图追踪你的成长轨迹

## 快速开始

### 方式一：直接本地使用

```bash
# 克隆仓库
git clone https://github.com/你的用户名/zhijian-kb.git
cd zhijian-kb

# 直接用浏览器打开（无需任何服务器）
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

### 方式二：GitHub Pages 在线部署（推荐）

1. Fork 本仓库，或将文件上传到你的新仓库
2. 进入仓库 **Settings → Pages**
3. Source 选择 `main` 分支 / `root` 目录 → Save
4. 约 1 分钟后访问：`https://你的用户名.github.io/zhijian-kb`

每次 `git push` 自动更新，免费且永久在线，手机也能访问。

## 配置 AI 功能

应用内置 Claude API 调用，可在剪藏时自动提炼摘要和标签。

1. 点击右上角 **⚙ 设置**
2. 填入 Anthropic API Key（格式：`sk-ant-…`）
3. 保存后，下次剪藏时会自动调用 Claude Sonnet 处理内容

> **隐私说明**：API Key 仅存储在本地 `localStorage`，不经过任何中间服务器，调用时直接从浏览器向 Anthropic API 发起请求。代码完全开源可审计。
>
> 没有 API Key 也可以正常使用所有功能，只是剪藏时没有 AI 提炼。

## Coze 文档导入

每天从 Coze 获取学习文档的标准操作（约 2 分钟）：

1. 打开 Coze，找到今日学习文档，**全选复制**（Ctrl+A → Ctrl+C）
2. 点击纸间顶部「快速剪藏」→ 切换到 **「Coze 文档」** 标签
3. 粘贴内容，点击「AI 处理并保存」
4. 在生成的笔记中补充 `[[双链]]`，关联已有知识

> **未来计划**：待 Coze 开放个人 Webhook，可配置自动推送，无需手动操作。

## 使用技巧

**习惯一：每篇笔记建立至少 1 个双链**
孤立的笔记是「知识坟场」。即使只写 `[[相关笔记名]]` 也比没有强。AI 相似推荐的准确度与双链密度正相关。

**习惯二：热点笔记设「热点」有效期**
AI 领域变化极快，三个月前的最佳实践可能已经过时。设置有效期后定期做「过期笔记清理」，防止知识库熵增。

**习惯三：每遇 AI 翻车案例立即归入「反模式」**
失败案例比成功案例更有学习价值。面试时能讲出「这个方向的坑」比只会说「我知道怎么做」印象深刻得多。

**习惯四：每隔 2-3 周写一篇「输出」维度笔记**
将某个知识点整理成可分享的内容，哪怕只有 200 字。写出来的过程会暴露真正不理解的地方。

## 常见问题

**Q：数据会丢失吗？**
数据存在浏览器 localStorage，清除缓存或换浏览器会丢失。建议将仓库托管到 GitHub 作为备份，并定期导出重要笔记。

**Q：没有 API Key 可以用吗？**
完全可以，只是剪藏时不会 AI 提炼。双链、图谱、雷达、日历等所有功能均不依赖 API Key。

**Q：手机上可以用吗？**
可以。部署到 GitHub Pages 后，手机浏览器可正常访问，侧边栏变为抽屉式，编辑器全屏展示。

**Q：能和 Obsidian 一起用吗？**
目前暂不支持直接同步，但你可以将 Obsidian 中的 `.md` 文件通过「快速剪藏 → Markdown」标签导入。双链语法完全兼容（均为 `[[]]` 格式）。

## 技术说明

- **完全无框架**：原生 HTML + CSS + JavaScript，无任何 npm 依赖
- **外部资源**：仅加载 Google Fonts 字体（可选，断网时回退系统字体）
- **AI 调用**：直接从浏览器调用 `api.anthropic.com/v1/messages`，模型为 `claude-sonnet-4-20250514`
- **数据存储**：`localStorage`，键名 `zhijian_v3`
- **兼容性**：Chrome 90+ / Safari 14+ / Firefox 88+（Edge 同 Chrome）
